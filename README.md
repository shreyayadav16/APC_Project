#  Advanced Calculator using Doubly Linked List
A command-line based big number calculator implemented in C using doubly 
linked lists to handle arithmetic operations beyond standard data type limits.
It features modular design, input validation, and efficient handling of signed 
numbers with dynamic memory management.
## Features
- Addition
- Subtraction
- Division
- Large number handling

## How to Run
gcc *.c
./a.out 23432+ 3421
